## 1. English Version

Here is a quick and concise Cheat Sheet for all Markdown syntax and formatting, designed cleanly and directly so you can find what your text engine needs at a glance:

### 1. Headings

Headings are created using the hashtag symbol `#` followed by a space:

```markdown
# Main Heading (H1)
## Subheading (H2)
### Third-level Heading (H3)
#### Fourth-level Heading (H4)
##### Fifth-level Heading (H5)
###### Sixth-level Heading (H6)

```

### 2. Text Formatting

Add emphasis, bold, or strikethrough effects to words:

```markdown
*Italic text* or _Italic text_
**Bold text** or __Bold text__
***Bold and Italic together***
~~Strikethrough text~~

```

### 3. Lists

#### Unordered Lists

Use an asterisk, plus, or minus sign followed by a space:

```markdown
* First item
* Second item
    * Sub-item (Press Tab)

```

#### Ordered Lists

Use regular numbers followed by a period and a space:

```markdown
1. First step
2. Second step
3. Third step

```

#### Task Lists

Create checkboxes and completion marks:

```markdown
- [x] Completed task
- [ ] Uncompleted task

```

### 4. Blockquotes

To highlight text or write deep, philosophical quotes:

```markdown
> This is a highlighted blockquote that appears beautifully in the content.
>> Nested blockquote (inside the main quote).

```

### 5. Code

#### Inline Code

To highlight a variable name or a small command within a sentence, wrap it in a backtick (```):

```markdown
Use the `openPopup()` function to trigger the default window.

```

#### Code Blocks

To write multi-line code blocks, use three backticks (`````) and specify the language name for syntax highlighting:

```javascript
function helloWorld() {
    console.log("Hello, World!");
}

```

### 6. Links & Images

#### Links

Place the text in square brackets and the URL in parentheses:

```markdown
[Click here to visit the site](https://example.com)

```

#### Images

Exactly the same as links, but add an exclamation mark `!` at the very beginning:

```markdown
![Image alt text](https://example.com/image.png)

```

### 7. Tables

To organize data and matrices quickly:

```markdown
| Item | Status | Rating |
| :--- | :---: | ---: |
| Architecture | Completed | 10/10 |
| Visual Style | Excellent | 9.5/10 |

```

> 💡 *Note: The colon `:` defines the alignment: left alignment `:---`, center alignment `:---:`, and right alignment `---:`.*

### 8. Horizontal Rules

For a complete visual separation between sections, use three or more asterisks, hyphens, or underscores:

```markdown
---
***
___

```

Save this guide in your coding notebook to be your quick reference whenever you are crafting Lore or documenting files.

## 2. Dari Version (لوازم راهنمای سریع مارک‌داون)

در این‌جا یک راهنمای سریع و مختصر (Cheat Sheet) برای تمام ساختارها و فرمت‌های زبان Markdown آورده شده است، که به شکلی منظم طراحی شده تا در یک نگاه به آنچه برای موتور متنی خود نیاز دارید، برسید:

### ۱. عنوان‌ها (Headings)

عنوان‌ها با استفاده از علامت هشتگ `#` همراه با یک فاصله (Space) ساخته می‌شوند:

```markdown
# عنوان اصلی (H1)
## عنوان فرعی (H2)
### عنوان سوم (H3)
#### عنوان چهارم (H4)
##### عنوان پنجم (H5)
###### عنوان ششم (H6)

```

### ۲. فرمت متون (Text Formatting)

افزودن تأثیرات کج، ضخیم و خط‌خورده روی کلمات:

```markdown
*متن کج* یا _متن کج_
**متن ضخیم (Bold)** یا __متن ضخیم__
***ضخیم و کج با هم***
~~متن خط‌خورده (Strikethrough)~~

```

### ۳. لیست‌ها (Lists)

#### لیست‌های غیرمسلسل (Unordered Lists)

از ستاره، علامت جمع یا منها همراه با یک فاصله استفاده کنید:

```markdown
* مورد اول
* مورد دوم
    * مورد فرعی (کلید Tab را فشار دهید)

```

#### لیست‌های مسلسل (Ordered Lists)

از اعداد معمولی همراه با نقطه و سپس یک فاصله استفاده کنید:

```markdown
1. قدم اول
2. قدم دوم
3. قدم سوم

```

#### لیست‌های وظایف (Task Lists)

ساختن چک‌باکس‌ها و نشانه‌گذاری آن‌ها:

```markdown
- [x] وظیفه تکمیل شده
- [ ] وظیفه تکمیل نشده

```

### ۴. نقل‌قول‌ها (Blockquotes)

برای برجسته کردن متون یا نوشتن نقل‌قول‌های عمیق و فلسفی:

```markdown
> این یک متن نقل‌قول برجسته است که به زیبایی در محتوا ظاهر می‌شود.
>> نقل‌قول تودرتو (داخل نقل‌قول اصلی).

```

### ۵. کدها (Code)

#### کد درون‌خطی (Inline Code)

برای برجسته کردن نام یک متغیر یا یک دستور کوچک در میان متن، آن را درون علامت باک‌تیک (```) قرار دهید:

```markdown
از تابع `openPopup()` برای باز کردن پنجره پیش‌فرض استفاده کنید.

```

#### بلوک‌های کامل کد (Code Blocks)

برای نگه‌داری خطوط کامل کد، از سه باک‌تیک (`````) همراه با نوشتن نام زبان برای رنگ‌آمیزی کد (Syntax Highlighting) استفاده کنید:

```javascript
function helloWorld() {
    console.log("Hello, World!");
}

```

### ۶. لینک‌ها و تصاویر (Links & Images)

#### لینک‌ها (Links)

متن را داخل دو قلاب `[]` و لینک را داخل دو کمان `()` قرار دهید:

```markdown
[برای بازدید از سایت این‌جا کلیک کنید](https://example.com)

```

#### تصاویر (Images)

دقیقاً مانند سیستم لینک‌ها، اما یک علامت ندای `!` در ابتدا اضافه کنید:

```markdown
![متن جایگزین تصویر](https://example.com/image.png)

```

### ۷. جدول‌ها (Tables)

برای تنظیم و سازماندهی سریع داده‌ها:

```markdown
| مورد | وضعیت | امتیاز |
| :--- | :---: | ---: |
| ساختار برده | تکمیل شده | 10/10 |
| سبک بصری | عالی | 9.5/10 |

```

> 💡 *یادداشت: علامت دو‌نقطه `:` جهت چیدمان را مشخص می‌کند: چپ‌چین `:---`، وسط‌چین `:---:`، و راست‌چین `---:`.*

### ۸. خطوط فاصل (Horizontal Rules)

برای جداسازی کامل بصری میان بخش‌ها، از سه یا چند ستاره، خط تیره یا خط زیرین استفاده کنید:

```markdown
---
***
___

```
